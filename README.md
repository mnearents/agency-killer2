# Agency Killer

A self-hosted marketing operations system that replaces a traditional marketing agency. Coordinates Meta ads, email/SMS, organic social, SEO/GEO, video/image production, and inventory awareness into a single platform with holistic cross-channel intelligence.

Built for e-commerce brands on Shopify.

## What It Does

- **Meta Ads** — Syncs campaign data, analyzes performance, generates strategic briefs with CTC framework integration
- **Video Production** — Indexes footage from Dropbox, transcribes audio (AssemblyAI), tags scenes (Twelve Labs), renders edited videos (Shotstack)
- **Blog Automation** — Generates SEO/GEO-optimized articles via Anthropic API, creates Shopify drafts, manages topic scheduling
- **Knowledge Base** — Syncs strategy documents from Dropbox, maintains system knowledge for AI context
- **Performance Data** — Correlates Meta ad spend with Shopify revenue, tracks subscription LTV, filters recurring orders
